Events - Map events and output binding

In this example, we demonstrate how to use the output bindings and event emitters. Mouse events are exposed using a pattern similar to (leafletClick), (leafletMouseDown), etc. The map Zoom/Move events are exposed using (leafletMapMove), (leafletMapZoom), etc. We also provide output bindings for the map center point - (leafletCenter), and the zoom level - (leafletZoom). 

<div leaflet style="height: 300px;"
        [leafletOptions]="options"
        [(leafletZoom)]="zoom"
        [(leafletCenter)]="center"
        (leafletClick)="handleEvent('click', $event)"
        (leafletMapMoveEnd)="handleEvent('mapMoveEnd', $event)"
        (leafletMapZoomEnd)="handleEvent('mapZoomEnd', $event)">
</div>
Zoom: {{zoom}}
Center: [{{center.lat}},{{center.lng}}]
